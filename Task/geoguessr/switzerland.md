# Switzerland

判断一个地方是否属于瑞士十分容易，因为瑞士具有独特的自然风光和相对明显的建筑风格，同时瑞士的旗帜在境内也随处可见。然而，判断某个地方的具体位置事实上十分困难。这里提供一些基础的瑞士地理相关知识
